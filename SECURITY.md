# Security Policy

## Security Baseline

CompText is a developer preview / local dry-run MVP. It is not a production security, compliance, or forensic product.

Please refer to our detailed security baseline and guidelines in [docs/COMPTEXT_SECURITY.md](docs/COMPTEXT_SECURITY.md) for more details.

## Reporting a Vulnerability

If you discover a potential security vulnerability, please do not open a public issue. Instead, report it to the maintainer.
