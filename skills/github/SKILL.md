# GitHub Skill

## Purpose

Guide GitHub review, pull request, and issue workflows.

## When to use

Use when inspecting review threads, preparing PR text, or responding to actionable comments.

## Required checks

- Inspect all review and Gemini threads.
- Fix actionable comments.
- Resolve completed threads only.

## Safety notes

- Do not create pull requests unless repository policy permits it.
- Do not merge, release, or enable auto-merge by default.
- Do not assume remote checks passed without verifying them.
