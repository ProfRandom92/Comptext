# Git Skill

## Purpose

Operate safely with local Git state, branches, diffs, and review preparation.

## When to use

Use for branch setup, status checks, diff inspection, and commit preparation.

## Required checks

- Confirm the current branch.
- Inspect `git status`.
- Review relevant diffs before summarizing or committing.

## Safety notes

- Do not run destructive Git commands without explicit approval.
- Do not auto-push or auto-merge unless the user explicitly asks.
- Branch new work from latest `main`.
